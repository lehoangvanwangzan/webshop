## Dừng process đang dùng cổng 3000
netstat -ano | Select-String ":3000" | Where-Object { $_ -match "LISTENING" } | ForEach-Object { $parts = ($_ -split '\s+').Trim() | Where-Object { $_ }; Stop-Process -Id $parts[-1] -Force }

## Dùng git fetch kết hợp git merger

git fetch origin  (Tải toàn bộ các thay đổi mới nhất từ server về nhưng chưa tác động vào code hiện tại của bạn)
git merge origin/master  (Áp dụng các thay đổi vừa tải về vào code hiện tại của bạn)
